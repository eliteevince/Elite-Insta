 ## Get Started
 
 #### 1. Clone the Repo
 
 On the command prompt run the following commands
 ```sh
 $ git clone https://github.com/Doha26/InstagramClone.git
 
 $ cd InstagramClone
 
 $ npm install
 
 $ react-native run-ios
 
 ```
